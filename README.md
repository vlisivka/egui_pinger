# egui_pinger

Simple pinger with ping time, jitter, and simple graph implemented in rust/egui.

Interface in Ukrainian.

Простий пінгувальник хостів написаний на Rust/egui, з часом пінга, розкидом, і простим графом.

Інтерфейс українською мовою.

## Building and installation

```
git clone https://github.com/vlisivka/egui_pinger && \
cd egui_pinger && \
cargo install --path .
```
